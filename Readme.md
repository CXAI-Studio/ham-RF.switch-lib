通常情况下，使用SMA接头的DPDT/SPDT射频开关功率耐受HF频段约300W，70cm波段约150W，23cm波段约100W，DPDT的耐受功率小于SPDT同插头规格，SPDT高功率版本，使用SC或者N头，HF频段耐受可超5kw，70cm波段耐受约2kw以内。仅为参考，详细内容请根据射频开关型号查找手册


一般射频开关的控制方式按照动作可以区分为自复位和磁保持型，控制电平又分为VCC/GND电平控制驱动或TTL电平驱动，部分磁保持内置续流二极管或相关元件，大部分自复位都需要在外部额外部署续流二极管，大功率SPDT和DPDT开关还需要根据情况使用吸收负载以减少反向电动势对于驱动电路的冲击。
